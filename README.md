# 飲食店検索アプリ（Restaurant Search App）

https://recipe-app-2025.vercel.app/

## アプリ概要
Reactを用いて作成した飲食店検索アプリです。  
自作のAPI（/api/search）を通じて店舗データを取得し、  
キーワード検索・一覧表示・詳細表示ができるSPAとして実装しています。

外部APIには依存せず、データ設計から検索ロジックまでを自分で実装しています。

---

## 主な機能
- 飲食店のキーワード検索（店名・ジャンル・エリア）
- 店舗一覧表示
- 店舗詳細ページ表示
- 画像付き店舗表示
- 検索語が未入力の場合のおすすめ店舗表示
